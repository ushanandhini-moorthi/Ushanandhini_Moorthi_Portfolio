# Markdown to HTML and PDF Conversion Scripts





### Method 1: Batch Script (Easiest)

**Step 1:** Open Command Prompt (CMD)

**Step 2:** Navigate to the conversion-scripts folder (inside `PrometheusObervability`)
```batch
cd "c:\Users\Sudhakar\Documents\Ushanandhini Moorthi\Portfolio\Ushanandhini_Moorthi_Portfolio\observability-writing-sample\PrometheusObervability\conversion-scripts"
```

**Step 3:** Run the conversion script
```batch
convert-markdown.bat "..\prometheus-to-otlp.md"
```
