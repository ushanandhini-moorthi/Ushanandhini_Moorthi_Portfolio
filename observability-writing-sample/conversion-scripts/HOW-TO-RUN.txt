# Markdown to HTML and PDF Conversion Scripts

This folder contains automation scripts to convert Markdown files to HTML and PDF formats with professional styling.

## Quick Start

### Method 1: Batch Script (Easiest)

**Step 1:** Open Command Prompt (CMD)

**Step 2:** Navigate to the conversion-scripts folder
```
cd "c:\Users\Sudhakar\Documents\Ushanandhini Moorthi\Portfolio\Ushanandhini_Moorthi_Portfolio\observability-writing-sample\conversion-scripts"
```

**Step 3:** Run the conversion script
```
convert-markdown.bat "..\PrometheusObervability\prometheus-to-otlp.md"
```

### Method 2: PowerShell Script

**Step 1:** Open PowerShell as Administrator

**Step 2:** Navigate to the conversion-scripts folder
```
cd "c:\Users\Sudhakar\Documents\Ushanandhini Moorthi\Portfolio\Ushanandhini_Moorthi_Portfolio\observability-writing-sample\conversion-scripts"
```

**Step 3:** Allow script execution 
```
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

**Step 4:** Run the conversion script
```
.\convert-markdown.ps1 -InputFile "..\PrometheusObervability\prometheus-to-otlp.md"
```
